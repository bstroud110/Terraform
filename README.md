High level README about the repository.

PURPOSE
To provide structured tasks for building environments in various cloud providers using Terraform.  This allows me to develop Terraform experience while simultaneously showcasing my progress and growth with this tool.

Folder structure is broken down by Cloud Provider (ie. AWS, GCP).  At present, within each folder will be Basic and Multi Region environments.

-Basic demo environment
    -Build just a very simple 3-Tier enviroment in AWS using one region
        -NGINX server in a public subnet
        -App server in a private subnet
        -RDS database with connectivity to the private subnet

-Multi region demo environment
    -Build a 3-Tier environment in AWS in multiple regions
        -NGINX server in a public subnet
        -App servers in private subnets
        -RDS database with connectivity to the private subnets
